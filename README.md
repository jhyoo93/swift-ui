📌 App
- App 객체는 SwiftUI 애플리케이션 구조 내 최상위 요소이며 애플리케이션의 실행 중인 각 인스턴스의 시작 및 생명 주기를 처리한다.
  또한 APP 요소는 애플리케이션의 사용자 인터페이스를 구성하는 다양한 Scene을 관리하는 역할을 한다. 애플리케이션에는 하나의 App 인스턴스만 포함된다.
